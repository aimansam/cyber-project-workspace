# Blog

Use this area for public-safe writeups, lab notes, and polished technical articles.

## Suggested Layout

```text
blog/
  drafts/
  published/
  templates/
```

## Good Blog Topics

- Building the local cyber lab.
- Red-team lessons from intentionally vulnerable targets.
- Blue-team detection engineering walkthroughs.
- Purple-team validation stories from lab exercises.
- Bug bounty methodology without exposing private target data.
- Tool comparisons and workflow notes.

## Publishing Rules

- Write from lab data or authorized findings only.
- Remove target names, tokens, cookies, credentials, customer data, emails, phone numbers, IP addresses, and other raw identifiers unless publication is explicitly approved.
- Replace sensitive screenshots with diagrams or sanitized recreations.
- Keep reproduction steps educational and scoped to defensive or authorized testing contexts.
- Store blog-ready images under `assets/images/`.
