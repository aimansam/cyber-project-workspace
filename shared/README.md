# Shared Assets

Use this area for project assets that support both red-team and blue-team work.

## Suggested Files

- `asset-inventory.csv`: approved systems and ownership.
- `severity-rubric.md`: impact and likelihood scoring.
- `evidence-naming.md`: naming convention for screenshots, logs, and exports.
- `report-template.md`: reusable finding format.
- `glossary.md`: project-specific terms and acronyms.

## Evidence Naming Convention

Use a predictable format:

```text
YYYYMMDD-project-target-artifact-description.ext
```

Example:

```text
20260525-lab-juice-shop-screenshot-login-bypass.png
```
