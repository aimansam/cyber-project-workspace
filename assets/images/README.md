# Uploaded Project Images

Use this area for project images you upload, such as blog hero images, sanitized screenshots, diagrams, and report visuals.

## Suggested Naming

```text
YYYYMMDD-project-short-description.ext
```

Examples:

```text
20260525-lab-dashboard-overview.png
20260525-red-team-recon-flow.jpg
20260525-blue-team-alert-timeline.webp
```

## Image Rules

- Do not store screenshots containing tokens, cookies, API keys, private URLs, emails, phone numbers, customer data, or other raw identifiers.
- Prefer sanitized diagrams or cropped screenshots for public posts when full screenshots would expose sensitive context.
- Keep original raw screenshots in `evidence/` only when storage is approved.
- Track source, license, and attribution for third-party images.
