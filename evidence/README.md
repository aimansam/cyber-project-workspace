# Evidence

Use this area for raw artifacts collected during authorized work.

## Suggested Layout

```text
evidence/
  lab/
  red-team/
  blue-team/
  screenshots/
  logs/
  pcaps/
```

## Handling Rules

- Keep raw evidence private unless sharing is approved.
- Redact secrets and personal data from report-ready copies.
- Preserve timestamps, target identifiers, and command context.
- Do not store credentials, tokens, cookies, or API keys here.
